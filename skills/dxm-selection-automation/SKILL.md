# 店小秘自动上架选品规则

## 默认任务规则

1. 价格区间默认值：5-20 USD，可按任务配置调整。
2. 目标数量默认值：100 个商品，可按任务配置调整。
3. 每个品类上限默认值：15 个商品，可按任务配置调整。
4. 同一品类内尽量保证商品多样性。
5. 同一款产品不同颜色允许保留。
6. 不允许重复采集完全相同的商品。
7. 默认跳过已经采集过的完全相同商品。

## 重复识别规则

允许保留：

1. 同一款产品不同颜色，允许作为不同商品分别上架。
2. 同一款产品不同颜色，视为不同 SKU，可以分别进入采集流程。
3. 同一款产品不同颜色，不属于重复商品，不允许过滤。

必须过滤：

1. 完全相同的 Amazon 链接。
2. 完全相同的 ASIN。
3. 完全相同的颜色、尺寸、套餐组合。
4. 同一商品被重复采集。
